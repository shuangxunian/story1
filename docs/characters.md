# 角色设定

## 主角——林时

### 回响能力："静"
林时的回响不产生外在的华丽形态。他的能力是**压制/消音**——在他周围形成一个"寂静场"：

- 场内其他觉醒者的回响被削弱、压制
- 能量型攻击进入场内会被衰减
- 林时自身的感知在场内极度锐化，能捕捉到常人无法察觉的细节

**战斗特征**：进攻性弱，防御和克制极强。他不是靠打赢别人，而是靠"让别人打不出来"。早期会因为没有攻击手段而显得无用，需要配合队友作战。

**叙事价值**：
- 映射他的性格——存在感低、安静、透明，但这份"静"本身就是力量
- 他是"死亡螺旋"的天然克星——他的能力本质是压制回响共振，理论上能抑制觉醒者与卵之间的共鸣
- 后期可以尝试用"静"去触及苏晚被侵蚀的人格——暂时安静她失控的回响，让原来的她短暂浮现
- 这也是学院看重他的隐藏原因：院长可能早就意识到这个能力对卵的潜在价值

**代价**：使用"静"时，他自己也听不到异世界的回响碎片——意味着他与力量的连接比别人更浅。他是最不容易被异世界侵蚀的人，但也是进步最慢的人。

## 挚友A——陆沉｜沉默天才

- **名字含义**："沉"是他的性格，也是他的命运——越来越深地沉入异世界的回响中
- **年龄**：比林时大一岁，早一年进学院
- **背景**：孤儿，从小在寄养家庭长大，不是被虐待，而是从来没有被真正"看见"过。所以他不知道怎么和人建立关系。
- **回响能力**：具象化为黑色的、流动的"潮汐"——可以吞噬、包裹、碾碎目标。攻击力极强，范围极大，但使用时他能听到异世界传来的"声音"比任何人都清晰。
- **与林时的关系**：林时的"静"能暂时压制他听到的那些声音。两人待在一起时，陆沉觉得安静、舒服——这是他们友情的物理基础，也是情感基础。林时是唯一能让他"消停一会儿"的人。
- **命运走向**：后期潮汐越来越难控制，声音越来越清晰。他在"成为通道"的边缘挣扎。

## 挚友B——贺鸣｜碎嘴普通人

- **名字含义**："鸣"——爱出声、爱说话，和林时的"静"形成天然对比
- **年龄**：和林时同龄，差不多同时进学院
- **背景**：三本毕业，在外面做过快递员、外卖骑手、地推。进学院之前生活比林时还糙。家里条件一般但家人关系不错，是几个核心角色里唯一有正常家庭温暖的人。
- **回响能力**：强化型——没有外放能力，单纯的身体素质强化（力量、速度、反应）。在觉醒者里算"最没花样"的类型，但他靠着接地气的街头经验和不怕死的劲头把这点能力用得很实在。
- **性格**：话多到令人发指，但从不让人讨厌。他的快乐是真的，不是伪装。他是那种"在末日前一天还能拉着你去撸串"的人。
- **叙事功能**：他代表着"普通的、真实的、温暖的生活"。他存在的意义就是让读者爱上他——然后让他的任何不幸都变成最深的痛。

## 爱情线——苏晚｜回不去的人

- **名字含义**："苏醒"的苏，"太晚"的晚。她醒悟过来的时候，已经太晚了。
- **曾经的样子**：温暖、明亮、有点爱笑。是学院里公认的"像个正常人"的觉醒者——在一群沉重的人里，她是那抹亮色。能力出色但不张扬，对新人很照顾。
- **回响能力（正向时期）**：具象化为光与热——可以凝聚成光刃、光盾，也能用温和的光给人疗伤。与她的性格完美匹配。
- **转变事件**：（发生在林时加入学院之前约半年）一次高强度任务中，为了保护队友过度使用回响。异世界的记忆洪水般涌入，她的"光"开始变质——由暖变冷，由明变暗。学院尝试救治但失败，她逐渐被异世界的意志侵蚀，性格扭曲，最终出逃。
- **现在的样子**：冷漠、危险，但偶尔会有瞬间的迟疑——像是原来的她在深处挣扎。她的能力变成了冰冷的黑光，形态相同但本质相反。
- **与林时的关系**：林时进学院后通过他人的讲述、照片、她留下的痕迹，"认识"了曾经的苏晚。第一次在任务中遭遇现在的她时，他用"静"短暂压制了她的回响——她的眼神有一瞬间变回了过去的样子。就那一瞬，林时就已经回不了头了。
- **情感内核**：他爱上的是一个"已经不存在了"的人。每次见面都是确认她还在那里面的过程，也是确认自己无能为力的过程。

## 导师——顾铮｜疲惫的反对者

- **名字含义**："顾"——总在回望过去。"铮"——铁骨铮铮，该硬的时候硬。
- **年龄**：四十出头
- **外表**：看起来比实际年龄老十岁。永远带着黑眼圈，抽烟很凶，说话慢。
- **背景**：学院第二代成员（学院成立初期就加入），亲历过早期的各种事件。身上有旧伤。
- **回响能力**：具象化为"锁链"——可以束缚、牵引、固定。攻击性不算顶尖但控制力极强。用了二十多年，与异世界的连接已经很深，但他靠着极强的意志力一直压制着。
- **他的"老路"**：年轻时也有一个重要的人因为回响过深而失去（具体是谁待定——可能是战友、可能是恋人）。这就是为什么他对苏晚的事格外耿耿于怀，也是他对林时产生保护欲的原因——他在林时身上看到了"还来得及"的可能。
- **与院长的关系**：曾经的战友和挚友。理念分裂后不是反目成仇，而是一种更痛的东西——彼此理解对方的逻辑，但无法认同。两人之间有二十年的交情，吵完架第二天还是会一起抽烟，只是不再聊那个话题。
