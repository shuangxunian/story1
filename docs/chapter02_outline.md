# 第二章：裂痕（扩展版·约80-100节）

## 总述
林时正式入编后进入常规外勤轮转。任务范围从本市扩展到周边城市。认知从"学院是正义的"开始松动——世界比想的复杂。本章核心：深入社会→发现更多诱发觉醒证据→首次遭遇苏晚→贺鸣父亲危机→陆沉恶化。

场景比例：社会外勤55% / 学院内30% / 个人生活15%

---

## 分段规划

### A. 正式成员的新日常（第1-8节）
- 001-002 → 入编后的变化：制服、权限升级（可进C栋部分区域）、任务频率增加、三人被分配到不同外勤小队但仍以A-2为主
- 003-004 → 第一次以正式成员身份出外勤——任务本身不大，但感受不同了。不再是"新编在学习"，而是"你要负责"
- 005-006 → 林时的"静"训练进入新阶段：开始练习对单一目标的定向压制（而非扇面），难度极大
- 007-008 → 日常碎片：给妈打电话（谎言在继续）、看望周瑞（周瑞开始训练了）、台阶夜聊（陆沉提到声音最近"多了一层"）

### B. 社会外勤·工厂线（第9-20节）
第一组重大社会任务——工厂流水线工人群体觉醒事件。
- 009-011 → 任务简报+出发：某制造业工厂近两周发生三起工人暴力事件（持刀、砸设备、殴打工友）。表面看像劳资纠纷+压力爆发，但检测到多点微弱回响波动——可能有觉醒者，也可能是大面积微量污染
- 012-015 → 进入工厂：伪装为劳动监察。流水线环境、工人群像（加班、欠薪、宿舍八人间）。沈默扫描发现不止一个回响源——至少三个工人有微弱觉醒迹象，且都是最近两三个月内
- 016-018 → 核心发现：三个工人的觉醒不是自然的——他们的宿舍里发现了类似A-12的物质痕迹（不是瓶子，是粉末残留在饮水机滤芯里）。有人在他们不知道的情况下给他们投放了诱发觉醒的药物
- 019-020 → 任务收尾+冲击：这不是个案。这是有组织的、针对特定群体的投放。林时开始系统性地记录——手机备忘录变成了一个越来越厚的"证据库"

### C. 贺鸣父亲（第21-26节）
- 021-023 → 贺鸣接到家里电话：父亲跟工地上的人打架，被打伤住院了。母亲在电话里哭。贺鸣第一次在林时面前崩溃
- 024-025 → 林时陪贺鸣请假回山东老家。用"静"近距离感知贺鸣父亲——确认微量污染。但无法告知家人真相，无法治疗
- 026 → 回程。贺鸣沉默了一路。林时不知道说什么。有些痛没有安慰的方法

### D. 陆沉的声音（第27-32节）
- 027-028 → 陆沉的异世界声音在恶化——不再是"背景噪音"，开始有了"内容"。他能听到模糊的词句，像是有什么东西在对他说话
- 029-030 → 一次训练中陆沉的潮汐短暂失控——黑色物质在食堂溢出了两秒。所有人都看到了。周教官事后单独谈话
- 031-032 → 陆沉开始主动找林时待在一起的次数变多。不说话——就坐在旁边。"静"的被动场能让他安静一点。林时意识到：自己对陆沉来说不只是朋友，还是"药"

### E. 社会外勤·高三学生（第33-42节）
- 033-035 → 任务简报：某重点高中高三学生，回响波动与情绪压力高度相关。每次模考前后波动剧烈
- 036-039 → 接触过程：比周瑞更复杂——这个学生没有被霸凌，压力来自"正常的"学业竞争。父母的期望、老师的关注、同学的比较。他的觉醒是纯粹被压力"压"出来的
- 040-042 → 任务的伦理困境加深：把一个正在备考的高三学生带走意味着什么？他的高考怎么办？他的人生怎么办？许岚："我们没有权力替他做这个选择。但我们也没有权力让他在不知情的情况下继续伤害周围的人。"

### F. A-12追踪（第43-50节）
- 043-045 → 林时整理手机备忘录中的所有A-12相关线索：张涛的瓶子（网上配方拼凑）、工厂饮水机的粉末（有组织投放）、不同案例中的共同特征
- 046-048 → 向顾铮提出自己的分析。顾铮态度从阻止变为默许——"你发现的东西——我已经知道了。但你自己发现和我告诉你，意义不同。"
- 049-050 → 关键信息：诱发觉醒不是意外——学院内部有人在做这件事。不是外部势力。是学院自己的人。顾铮："你开始碰到那条线了。小心。"

### G. 苏晚·初遇（第51-65节）
本章最重要的段落——苏晚首次正面出场。
- 051-053 → 一次追踪任务：C区连续出现不规则的高强度回响波动，特征与已知的所有觉醒者都不匹配。许岚带A-2前往
- 054-057 → 追踪过程：城市边缘的废弃工业区。沈默的感知被干扰——对方的回响有主动规避检测的能力。逐步逼近
- 058-060 → 遭遇：她出现了。黑色的光——形态像是曾经的光刃，但颜色完全相反。冷的、暗的。许岚认出了她。"苏晚。"所有人的表情都变了
- 061-063 → 战斗/对峙：苏晚的战斗力远超A-2的应对能力。许岚命令撤退。但林时在混乱中释放了"静"——苏晚的黑光被压制了。她停了下来。她的眼神——有一瞬间——变了。那个笑得很开的短发女孩。就那一瞬间。然后她消失了
- 064-065 → 余波：许岚让所有人保密。"苏晚的事——不能上报告。"林时问为什么。许岚："因为如果上报——总部会派清除组来。那就不是追回——是击杀。"

### H. 裂痕显现（第66-75节）
- 066-068 → 林时消化苏晚的冲击。通过陆沉了解更多苏晚的过去——陆沉终于愿意多说一些了
- 069-071 → 林时在C栋查找苏晚的旧档案——宋清帮了忙（冒了风险）。档案记录：苏晚的回响曾经被评为"最具正向稳定性"的案例之一。她的转变来得突然且无预警
- 072-074 → 多条线索汇聚后林时的认知开始动摇：诱发觉醒是学院在做→苏晚的转变可能也与学院有关→学院到底在保护人还是在制造问题？
- 075 → 第二章终节：林时的手机备忘录不再叫"观察"了。他新建了一个备忘录。名字叫"裂痕"

---

## 本章种下的种子
1. A-12的投放是有组织的——来自学院内部（第四章揭示全貌）
2. 苏晚的存在和她的"一瞬间"——林时无法放下（第七章主线）
3. 贺鸣父亲的微量污染——无法治疗的无力感（第八章危机伏笔）
4. 陆沉的声音开始有"内容"——异世界在对他说话（第三章、第九章）
5. 许岚隐瞒苏晚——她在保护苏晚还是在保护自己？
6. 林时开始系统性记录证据——从被动接受到主动追查
7. 觉醒者群体投放——不止个案，是系统性行为
