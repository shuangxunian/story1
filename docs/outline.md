# 大纲 v0.2

## 一、世界观

### 表层世界
现代社会，一切如常。社会内卷严重，学业压力、就业压力、生存压力是所有人的日常。

### 里层世界
多年前（具体时间待定），异世界曾短暂入侵现实世界，入侵失败后留下了"残留"——一种无形的污染源。污染源会感染人类，引发觉醒/变异。

入侵者还留下了一样东西：**卵**。

### 卵
被学院封存在地下深处。没有人知道它是什么——武器、生命体、还是通道。它唯一可观测的行为是"跳动"，像心跳一样。最早被发现时，频率是一年一次。最近几年，频率开始加快。这是学院高层变得激进的直接原因——他们不知道卵孵化/激活意味着什么，但他们知道时间不多了。

### 污染的三种层次
- **正向污染**：纯粹的属性增强（力量、感知、特殊能力），人格保持完整。少数人。
- **负向污染**：属性增强的同时，性格被扭曲、引入歧路，变得极端、暴戾或冷酷，走向对立面。少数人。
- **微量污染**：没有获得任何能力增强，仅情绪被侵蚀——莫名的易怒、抑郁、偏激、戾气上涌。受害者本人完全不知道原因，以为是自己的问题、社会的问题、时代的问题。占绝大多数。

大部分人的觉醒是无声无息的，自己都不知道是怎么回事。

### 社会氛围的暗变
微量污染的广泛扩散导致整个社会的氛围在肉眼可见地恶化：暴力事件增多、网络戾气加重、路怒、校园霸凌、职场PUA频发。人们只觉得"今年怎么了"、"大环境不好"，没有人知道真正的原因。这种"时代的病"是故事的底色，也是学院最大的罪——他们释放污染的代价，由千千万万无辜的普通人在无知中承受。

### 学院（组织名称待定）
维持里世界秩序的核心组织。表面上是在保护普通人、铲除恶性变异者。

**核心秘密（大真相）**：学院本身就是污染源的主动释放者。他们有计划地将异族污染源散布到人群中，筛选出正向觉醒者纳入麾下，铲除负向变异者。

**院长的心路历程（A→发现→骑虎难下）：**

院长一开始是真心想救人类。卵在加速，他做了自认为唯一能做的事——扩招觉醒者、积攒战力。他不是坏人，是那种"宁可背负骂名也要把事做了"的人。

直到导师发现了异常：感染的人越多，卵跳得反而越快了。导师和院长大吵一架，要求停下来。院长最终被说服，给了一个月观察期——停止释放污染源，观测卵的变化。

观察结果证实了最坏的猜想：觉醒者的回响确实在反哺卵，他们一直在喂养它。

**但问题是：已经晚了。**

增速已经上去了——类似导数的概念，就算现在停止扩招，已有的觉醒者数量足以维持卵的加速。曲线已经过了拐点，刹不住车了。

这把院长逼入了一个绝境式的逻辑：
- 停下来？卵照样加速，但己方战力不足，等卵孵化时毫无还手之力
- 继续扩招？虽然在喂养卵，但至少"那一天"到来时手里有兵
- 他选择了继续——"既然已经在错误的路上走了这么远，回头等于白死，只能赌向前走能走到底"

这不是疯狂，是一种已经无路可退的悲壮。明知道自己在加速末日，但手里只有这一张牌。

**内部分歧**：导师从此与院长理念彻底分裂。导师认为应该停下来寻找其他可能性，院长认为没有时间了。两人都没有错，也都没有对。（详见导师角色）

## 二、力量体系——"回响"

污染源残留在人体内形成**"回响"**——异世界的碎片记忆与人类意识产生共振。

### 能力具象化
回响会根据每个觉醒者的个人特质（性格、经历、内心深处的渴望或恐惧）自动生成最适合的战斗形态。不是你选择能力，是能力选择你。

具象化的形式类似于一种"气"的凝聚——可以构成攻击手段（武器、弹射物、冲击波）、防御手段（护盾、铠甲）、或特殊能力（感知、操控、强化），因人而异。每个人的回响形态独一无二，没有两个完全相同的觉醒者。

### 核心规则
- 不设严格的等级阶梯，以"能做什么"和"承受了什么代价"来体现强弱
- 使用能力时会短暂"听到"或"看到"异世界的片段——声音、画面、情绪
- **代价与隐患**：回响用得越多，与异世界的连接越深。过度使用者可能成为异世界重新入侵的通道——这是力量的天花板，也是悬在每个觉醒者头上的剑
- 这也解释了负向污染的本质：他们的回响失控，被异世界的意志/记忆侵蚀了人格

### 与"卵"的关系——死亡螺旋
卵与所有觉醒者体内的回响存在共振关系。这构成了一个致命的反馈循环：

**卵跳动加速 → 学院恐慌，加大污染释放、批量制造觉醒者 → 更多觉醒者的回响与卵共振 → 卵跳得更快 → 学院更激进 → 循环加速……**

学院以为自己在备战，实际上在亲手喂养那颗卵。

卵就是这个故事的时钟——它跳得越快，结局越近。它的跳动频率就是故事的倒计时。

这个真相是整个故事最核心的反转之一：
- 导师可能隐约直觉到了这一点，但缺乏证据
- 院长是否知道？如果知道还在做——他的动机将更加复杂和可怕
- 林时最终发现这个循环时，将面临最残酷的选择

## 三、主角——林时

- **姓名**：林时
- **身份**：普通二本，计算机专业，大四
- **现实处境**：费尽全力找到一家小公司的实习，做运维，工位在角落，存在感极低。生活拮据，前途迷茫。
- **性格**：安静但内心戏丰富，自嘲式幽默，口头禅"差不多得了"。表面低欲望自我保护，骨子里有股不认命的劲。不是倒霉蛋，是普通到几乎透明的人。
- **觉醒契机**：实习加班到凌晨，独自走夜路回出租屋，撞见学院的"清除行动"——成员正在追杀一个负向变异者。变异者濒死之际释放的污染波及林时，林时被迫觉醒。学院本想抹除记忆，发现他是正向污染，将其带走。
- **成长弧线**：从"只想活好自己的普通人"→"被卷入里世界的新人"→"开始质疑学院"→"发现大真相"→"不得不做出选择的人"

## 四、核心配角

### 挚友A——（名字待定）｜沉默天才
- 比林时早进学院，回响能力极强
- 极度沉默寡言，不是冷酷，是真的不知道怎么社交
- 林时是第一个拿他当正常人对待的人，两人因此建立深厚友情
- **隐患**：回响过深，与异世界的连接越来越强，后期面临"被异世界吞噬/成为通道"的危机
- 他的危机是"回响"体系最直观的体现——力量的尽头不是更强，而是失去自我

### 挚友B——（名字待定）｜碎嘴普通人
- 和林时差不多时间进学院，回响能力平庸
- 性格极其外向、话多、自来熟，是团队的气氛担当
- 和林时因为"都是菜鸟"结盟，是林时在压抑世界里最重要的情绪出口
- **叙事价值**：如果他后期出事（受伤/牺牲/被污染），对读者的杀伤力巨大，因为他代表的是"普通的快乐"

### 爱情线——（名字待定）｜回不去的人
- 曾经是学院的正向觉醒者，能力出色
- 因为某个事件（待定），回响失控，转向了负向污染，性格被扭曲
- 如今站在对立面，成为了学院的清除目标之一
- 林时知道她原来的样子——或许是进学院早期相处过的人，温暖的、明亮的
- 每次交手都是情感撕裂：她已经不是她了，但林时无法把她当作单纯的敌人
- 与"回响"体系深度绑定：她就是"用得太多、走得太深"的活生生的例子
- **情感内核**：不是"追不到"，而是"回不去"。她曾经就在身边，现在隔着一整个世界

### 导师——（名字待定）｜疲惫的反对者
- 学院资深成员，经验丰富，见过太多
- **关键事件**：他是第一个发现"觉醒者越多，卵跳越快"这个关联的人。与院长大吵一架，逼出了一个月的观察期，结果证实了他的猜想——回响在反哺卵
- **当下立场**：观察期之后，院长选择继续扩招。导师无法接受这个决定，认为应该停下来寻找其他可能性，而不是在一条已知错误的路上越走越远。但他也拿不出"其他可能性"到底是什么。
- **困境**：他知道院长的逻辑不是疯狂——确实已经刹不住车了。但他的直觉告诉他，继续下去只会让一切更快崩塌。反对，却说不出替代方案，这让他的反对显得无力
- 对林时有一种复杂的情感：看到了年轻时的自己，想保护他，又知道保护不了
- "别走我的老路"——他的老路是什么？（待定，可能是他也曾失去过重要的人）
- 他的存在让学院内部不是铁板一块，为后期学院分裂/内战埋下种子

## 五、叙事结构（原神式）

### 大目标
追寻真相——我为什么会觉醒？学院到底在做什么？卵是什么？那个即将到来的终点在哪里？

### 小目标（每卷/每篇章）
每个篇章有独立的：
- 事件（任务/冲突/谜团）
- 配角（有自己故事的人）
- 势力（不同立场的组织或个人）
- 情感线（友情/爱情/羁绊）
- 真相碎片（拼向大真相的一块拼图）

### 认知递进线
1. "学院是正义的，我是被拯救的人"
2. "这个世界比我想的复杂，敌人不全是坏人"
3. "学院本身就是灰色的，他们在制造灾难"
4. "学院的动机是恐惧，卵在加速，没人知道终点"
5. "也许没有谁是对的，但必须做出选择"

## 六、情感基调

- **主基调**：压抑、沉重，现实的重量不会因为力量而消失
- **亮色**：极少但珍贵的温暖——碎嘴挚友的玩笑、导师沉默的保护、回忆中她还没有变的时候
- **参考感觉**：龙族中路明非与上杉绘梨衣的关系——得不到、回不去、但那段时光是真实的
- **情感线**：友情、爱情、亲情都会涉及，不偏废
