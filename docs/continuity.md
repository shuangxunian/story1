# 连续性追踪

## 已确立的世界规则
- 污染分三种：正向（增强+人格完整）、负向（增强+人格扭曲）、微量（无能力+情绪恶化）
- 回响：异世界碎片与人类意识共振产生的能力，形态因人而异
- 回响使用过度会加深与异世界的连接，可能被侵蚀
- 卵在学院地下，跳动频率在加速
- 觉醒者的回响与卵存在共振关系（死亡螺旋）
- 学院主动释放污染源扩招觉醒者
- 院长最初不知道死亡螺旋，后被导师说服观察一个月后发现，但已无法回头
- 觉醒者无法回归正常生活——回响会影响周围普通人（顾铮告知林时）
- 学院有记忆消除技术
- 负向变异频率在加速增长（陆沉统计：6年内从月均3-4起增至12-15起，非线性增长）
- 诱发觉醒的定向投放已确认为系统性行为——三年37起，其中定向投放16起，14起投放者未查明
- 投放目标集中于底层劳动群体（工厂/工地/城中村）
- 正式入编后权限升级：C栋一二层（情报室+资料室）、终端可查已结案任务报告
- A-12变种在恒远工厂被确认：87%重合度+13%新型缓释剂（滤芯中持续释放两周）
- 工人暴力行为可能与A-12诱发觉醒前期症状相关（情绪不稳、易怒、睡眠障碍）
- A-12诱发的早期觉醒有可能在停止接触后自行消退
- 陆沉的异世界声音出现清晰词汇："回来"——重复多次
- 周瑞的回响为"风"——风场感知型偏向，五米范围，训练中
- 前驱阶段触发模式分两种：压力触发型（外部压力下波动出现）和放松/潜意识触发型（白天回响被压制，夜间意识松弛时泄漏）
- "空域梦"：觉醒前驱中的特定梦境——极高处/极空旷处/极安静处，潜意识在"校准"回响频率
- 前驱强度分级（完全觉醒=100）：极早期(1-5)无威胁无泄漏，阈值10考虑调岗/15必须离开危险环境/20需带回
- 单点压制距离衰减非线性：5m=93%、10m=78%、15m=61%、20m=42%
- 回响-声音关系演化四阶段：互斥→并行→共振（陆沉已进入共振——声音与回响同频同步）
- 双通道输出（扇面+单点同时不同方向）：消耗远超单一模式，13秒极限，核心频率降14%
- 微量污染已扩散至普通居民区（林时家小区楼下老王——退休后脾气暴变、失眠、头里有声音）
- "静"在无回响环境中特别安分——环境对回响活跃度有影响
- 社区排查机制：外勤组例行扫描，微量污染固定源仅标记坐标入库，由"上面"决定后续
- 训练恢复机制：核心频率反复升降产生微损伤，定期停训让身体修复
- CR-53内部分析报告：六年37起A-12案例，所有样本化学指纹指向同一源头，跨越8个据点
- 顾铮四年前提交的A-12分析报告被总部情报处拦截——"已知悉。不建议继续追查"
- 学院可能为维持觉醒者数量而默许A-12投放——"如果自然觉醒不够快，就人工制造"
- 年假制度：三天内许岚签，超三天需顾铮或更上面签
- 赵师傅的"空域梦"——极早期前驱（强度3/100），两周一次监测
