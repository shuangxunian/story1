# 故事内时间线

## 远古
- 异世界短暂入侵现实世界，入侵失败
- 残留物（污染源）和"卵"被留在了现实世界

## 数十年前（具体年份待定）
- 学院成立，封存卵，开始研究污染和觉醒现象
- 顾铮年轻时加入学院（第二代成员）

## 数年前
- 卵的跳动频率开始从"一年一次"变快
- 院长决定主动释放污染源，加速筛选觉醒者

## 约一年半前
- 导师顾铮发现"觉醒者越多→卵跳越快"的关联
- 顾铮与院长大吵，争取到一个月观察期
- 观察结果确认死亡螺旋的存在
- 院长认为已无退路，选择继续扩招
- 顾铮与院长理念彻底分裂

## 约半年前
- 苏晚在一次高强度任务中过度使用回响
- 回响失控，由正向转为负向，性格被侵蚀
- 学院救治失败，苏晚出逃
- 苏晚被列入清除名单

- 林时大四，在小公司做运维实习
