# 第一卷：觉醒

## 总述
林时从一个为实习焦头烂额的普通大学生，被意外拖入里世界。本卷的核心任务是：建立共鸣（让读者觉得"这就是我的生活"）→ 打破日常（觉醒事件）→ 进入新世界（学院）→ 种下所有重要关系的种子 → 第一次任务中埋下疑点。

## 章节规划

### 第一章：差不多得了
林时的日常。早上挤地铁，地铁上有人因为被踩了一脚破口大骂（微量污染的社会底色）。到公司，做着枯燥的运维工作，工位在角落。午饭吃最便宜的外卖。和同事没什么交集。晚上加班，leader说"年轻人多学学"。

穿插他的内心独白——自嘲、幽默、但偶尔流露出真实的疲惫。他不是怨天尤人，只是觉得生活像一杯温吞水，不烫也不凉，就是没什么味道。

收尾：加班结束，凌晨一点，他走在回出租屋的路上。街上几乎没人。他塞着耳机，想着明天还要早起。

### 第二章：巷子
林时抄近道走进一条小巷，听到异响。他看到了不该看到的东西——

一个人（负向变异者）在巷子尽头像困兽一样被围着。两个穿着普通但动作异常的人（学院清除组）正在"处理"他。变异者浑身冒着不正常的黑气，发出不像人能发出的声音。清除组的手上凝聚着某种能量。

林时本能想跑，但变异者在濒死之际爆发出一波污染冲击。冲击波扫过林时——

他的意识陷入短暂的黑暗。黑暗中，他"听到"了什么。不是声音，更像是整个世界在某个频率上震颤了一下。然后，静了。

一种前所未有的"安静"从他体内扩散出来。清除组的能量在他附近突然变得不稳定、被削弱。所有人都愣住了。

### 第三章：你不该在这
清除组成员意识到林时觉醒了——而且是正向污染。他们控制住局面后，有两个选择：抹除记忆或者带走。但林时的能力（压制了他们的回响）让他们觉得这个人"有点特殊"。一个电话之后，决定带走。

林时被带到一个看起来像是老写字楼的地方（学院的城市据点）。简单的体检、登记。他全程懵的。一个疲惫的中年男人走进来，点了根烟，看了他一眼："你叫什么？"

这是顾铮的第一次出场。没有任何戏剧化的台词，就是一个看起来压力很大的中年人在做他的工作。

章末：林时被安排在据点里住下，被告知"明天有人跟你解释"。他躺在陌生的床上，盯着天花板。耳机里的歌还在放，但他已经什么都听不进去了。

### 第四章：另一个世界
顾铮向林时解释了基本情况：污染、觉醒、正向与负向、学院的存在（只说了表面——保护普通人、清除变异者）。没有说卵，没有说主动释放。

林时的第一反应不是热血沸腾，而是："那我的实习怎么办？"

顾铮愣了一下，然后差点笑出来。

这一章的重点是：林时的"普通人"心态与里世界的荒诞形成对比。别人觉醒后可能想着拯救世界，他想的是自己还没交的房租和实习转正的事。

章末：林时同意"先了解一下"，被安排进入学院正式据点（更大的基地/总部）。

### 第五章：菜鸟
林时到达学院。这里不像学校也不像军营，更像一个气氛沉闷的大型机构——每个人都很忙，每个人看起来都不太开心。

他在新人登记处遇到贺鸣——另一个刚被"捡回来"的觉醒者。贺鸣话多得像机关枪，五分钟之内林时就知道了他的血型、前女友数量、和他觉得最好吃的螺蛳粉品牌。

林时原本想保持距离，但贺鸣根本不给他这个机会。两人被分到同一个新人小组。

### 第六章：安静的人
新人训练开始。训练主要是学会感知和控制自己的回响。

林时发现自己的能力很尴尬——别人在练习释放能力，他的能力是"让别人释放不出来"。训练场上，他站在中间，周围一圈人的能力都被削弱了。教官觉得这不好测试，把他安排到一边单独练。

贺鸣过来安慰："没事兄弟，我也就一个浑身变硬的功能，你说我是不是该去拍个短视频？"

这一章引入陆沉。他在训练场的角落，独自练习，黑色潮汐一波一波地涌出来，其他新人都离他远远的。不是因为他可怕——好吧，确实有点可怕——而是他身上那种"别靠近我"的气场。

林时无意间走近了陆沉的训练范围。陆沉的潮汐在接近林时时明显减弱了（"静"的效果）。陆沉停下来，看了他一眼。

那一眼没有敌意，也没有感谢。只是一种很轻的意外，像是在说："哦，安静了。"

### 第七章：她曾经在这里
林时在学院里逐渐适应。通过日常训练和任务简报，他开始了解这个世界的规则。

穿插细节：他注意到学院里某些地方有一个名字被提起时，人们的反应很微妙——不是恐惧，而是一种混合了惋惜和回避的沉默。那个名字是苏晚。

他在休息室的公告板上看到一张被遮住了大半的旧照片——几个觉醒者的合影，其中一个女孩在笑，笑容明亮到和周围的氛围格格不入。有人（可能是贺鸣从别人那听来的）告诉他：那是苏晚，半年前还在的时候，是学院里大家最喜欢的人。后来出了事，现在是"清除名单"上的人了。

林时没多问。但那张照片上的笑容，他记住了。

### 第八章：第一次任务
林时、贺鸣跟随顾铮的小队执行第一次外勤任务——处理一个在城市边缘出现的负向变异者。

顾铮把任务定义为"带新人见世面"，让林时和贺鸣跟在后面观察。

实战比训练残酷得多。变异者曾经是一个普通人（由周围的个人物品和生活痕迹可以推断出来），现在已经完全失控。顾铮的锁链冷静地制服了对方。过程中没有热血的台词，只有压抑的效率。

任务后，清理现场。林时看到变异者住处的照片、外卖单、一个翻开的手机屏幕上停留着的求职App。这个人曾经和他一样。

顾铮抽着烟站在旁边，什么都没说。

### 第九章：不对劲
任务后的例行汇报中，林时注意到一个细节：任务简报上说这个变异者是"自然觉醒后负向化"的，但现场的某些痕迹（待具体设计——比如一个不自然的注射痕迹、一瓶来路不明的药物、或者变异者的异变模式与自然觉醒不太一样）让他觉得不对劲。

他试着跟贺鸣提了一句，贺鸣没当回事："你想多了，可能就是个巧合。"

他又去问顾铮。顾铮的手停顿了一秒——只有一秒——然后说："新人不要瞎琢磨任务以外的事。"

但那一秒的停顿，林时看到了。

### 第十章：安静的人们
本章做两件事——

第一，深化林时和陆沉的关系。一次偶然的机会，林时发现陆沉一个人坐在天台上，用手捂着耳朵。林时走过去坐在旁边什么都没说，但他的"静"自然覆盖了陆沉。陆沉的手慢慢放下来。

两人之间的第一段真正的对话非常短：
陆沉："你能一直待在这吗？"
林时："……行吧。"

第二，埋下一条暗线。林时在学院的某个不该去的区域迷路时，感觉到了一种奇怪的"跳动"——不是听到的，而是他的能力感应到的。一下。很远。很深。像心跳。

他没有告诉任何人。

### 尾声：留下来
林时考虑了几天后，正式决定留在学院。他给实习的公司发了辞职消息，对方只回了一个"好"。

没人在意他去了哪里。这个世界就是这样的。

他把手机扔到床上，看着天花板想：至少这里有几个觉得我有用的人。

第一卷完。

## 本卷种下的种子（后续篇章使用）
1. 苏晚的存在已经被建立——通过照片和他人口述，但尚未正面出场
2. 林时感应到了"卵"的跳动，但不知道那是什么
3. 变异者现场的"不对劲"是主动污染的线索，种下了对学院的第一颗怀疑种子
4. 顾铮的一秒停顿暗示他知道些什么
5. 陆沉的"声音"问题为他后期的危机做了铺垫
6. 贺鸣的正常家庭和温暖性格被充分建立——为后续他的命运变化储备情感势能
