# 第一章：觉醒（扩展版·约120节）

## 总述
林时从普通实习生到正式加入学院。本章核心：建立共鸣→打破日常→进入新世界→建立关系→第一批疑点→做出选择。

场景比例：学院内30% / 社会外勤50% / 个人生活交叉20%

## 现有20节定位
已写的20节作为骨架保留，在关键位置之间插入扩展节。

---

## 分段规划

### A. 觉醒前（第1-5节）[已有001-001，扩展至5节]
- 001 ✅ 林时的一天（实习、地铁、加班、夜路）
- 002-003 [新增] 往前补一两天的日常：室友关系、投简历被拒的记忆、和母亲的日常通话、便利店买泡面时遇到的小摩擦（社会戾气底色）
- 004 [新增] 觉醒当天的白天——一个更完整的"最后的普通日"，公司出了个小事故让他加班更晚
- 005 → 原002前半，走进巷子前的最后几分钟

### B. 觉醒事件（第6-12节）[已有002，扩展至7节]
- 006-007 → 原002拆分扩展，巷子里的目击过程放慢，增加感官细节和心理活动
- 008 → 觉醒的瞬间，第一次感受到"静"，更细腻地写那种从未有过的安静
- 009 → 清除组的反应，冲锋衣女人（许岚）的处理方式，被带上车
- 010-011 → 到达据点，等待，环境描写，孤独感
- 012 → 顾铮出场，初次对话

### C. 据点过渡（第13-22节）[已有003-004，扩展至10节]
- 013-015 → 据点里的第一夜+第二天早晨，更多据点的日常细节（其他工作人员、墙上的地图和标记）
- 016-018 → 顾铮的解释拆成多次对话，不是一次灌完，穿插林时自己的消化和追问
- 019 → 林时的第一个选择：去总部还是消除记忆。不是立刻决定，给他一天的犹豫时间
- 020 → 犹豫日：他回了一趟出租屋（在监护下），看到室友、看到自己的生活，对比
- 021 → 发辞职消息，给母亲打电话（扩展原005的内容）
- 022 → 被送往总部

### D. 进入学院（第23-40节）[已有005-006，扩展至18节]
- 023-025 → 到达学院，第一印象，环境描写（不是校园是工业园区），登记、入住
- 026-028 → 遇到贺鸣，两人从陌生到熟悉的过程不要一节完成，给2-3节慢慢展开
- 029-031 → 学院的日常生活：食堂、宿舍楼、作息制度、其他成员的群像速写
- 032-034 → 第一周的适应——不适感、信息过载、对这个地方的观察
- 035-037 → 贺鸣的背景故事展开：送外卖的经历、家庭情况、他为什么愿意留下来
- 038-040 → 林时独处的时刻：给母亲的微信、夜晚失眠、开始写手机备忘录记录观察

### E. 训练+陆沉（第41-60节）[已有007-008，扩展至20节]
- 041-044 → 新人训练的完整展开：教官的态度、训练内容、其他新人的群像（不止七个人，可以有更多面孔）
- 045-048 → 林时的"静"在训练中造成的尴尬，被隔离训练，内心的挫败感
- 049-051 → 首次与陆沉的回响互动（原007核心场景），但之后不是立刻交朋友，而是各自保持距离
- 052-054 → 关于陆沉的侧面信息收集（通过贺鸣、食堂闲聊、其他学员的态度）
- 055-057 → 食堂事件（贺鸣拉陆沉坐下），三人第一次同桌但还很生疏
- 058-060 → 多次训练中的渐进互动，陆沉开始主动靠近林时的"静"范围

### F. 苏晚的影子（第61-70节）[已有009，扩展至10节]
- 061-063 → 学院里关于苏晚的线索是分散的：一次无意中听到的对话片段、一个空着的宿舍房间、休息室照片
- 064-066 → 贺鸣打听来的信息，分几次告诉林时，每次多一点
- 067-068 → 从其他老成员口中侧面了解苏晚的能力和性格
- 069-070 → 顾铮在某个不经意的瞬间对苏晚的名字有反应（又一个微小的线索）

### G. 外勤任务群（第71-95节）[已有010-011+017，扩展至25节]
这是第一章最重要的扩展区域——社会场景。

**任务一：城中村·张涛（约8节）**
- 071-073 → 任务准备、出发、路上顾铮的只言片语
- 074-076 → 任务过程（扩展原010），张涛的生活痕迹用更多篇幅去写
- 077-078 → 任务后的沉默，林时发现瓶子，向贺鸣和顾铮追问（原011）

**任务二：工地·打架事件（约5节）[新增]**
- 079-081 → 不是去清除变异者，而是一次"排查"任务——工地上接连发生暴力冲突，学院去调查是否有觉醒者。结果发现只是微量污染。没有人需要被清除，但所有人都在变坏。
- 082-083 → 林时在工地上看到的普通人的困境，和贺鸣父亲的工地生活形成对照

**任务三：商业区·刘芳（约8节）**
- 084-087 → 扩展原017，刘芳的半变异，林时的"静"起作用
- 088-089 → 刘芳被带回学院后的处理，林时的追问
- 090-091 → 顾铮告诉林时"没有人回过家"（原018扩展）

**任务四：学校附近·学生（约4节）[新增]**
- 092-093 → 一所中学附近出现回响波动，排查后发现是一个初中生在校园霸凌中觉醒了——正向的，但极度恐惧和不安
- 094-095 → 这个孩子被带走的过程让林时非常不舒服——他只有十四岁

### H. 深化关系+暗线（第96-110节）[已有012-013+014-016，扩展至15节]
- 096-098 → 天台场景扩展（原012），多次天台独处，陆沉慢慢打开

### I. 选择+第一章收尾（第111-120节）[已有019-020，扩展至10节]
