# 世界设定

## 地理

- **主要场景**：现代中国某城市。学院总部（A栋研究/B栋训练/C栋情报/D栋训练）、城区各片区（城南/城西/城东/城北废弃工业区）
- **气候**：四季分明，无特殊设定
- **地理限制**：学院隐藏于城市中，普通人不可见。废弃工业区为负向觉醒者聚集地

## 政治

- **统治形式**：学院是维持里世界秩序的核心组织，对外隐蔽运作
- **主要势力**：学院（官方）、地下组织（负向觉醒者自组织，陈渡为核心）、微量污染下的普通社会
- **社会分层**：正向觉醒者（学院成员）→ 负向觉醒者（清除目标/地下组织）→ 微量污染者（不自知的普通人）→ 未受污染普通人
- **不服从的代价**：负向觉醒者被"清除"；知情普通人被记忆消除；内部异见者被边缘化（顾铮的报告被拦截）

## 规则

### 世界级
- 异世界曾短暂入侵现实，失败后留下"残留"污染源和一枚"卵"
- 卵被封存在学院地下，跳动频率持续加速（死亡倒计时）
- 污染三层次：正向（能力+人格完整）、负向（能力+人格扭曲）、微量（无能力+情绪恶化）

### 社会级
- 微量污染广泛扩散导致社会氛围恶化（暴力增多、戾气加重），普通人不知原因
- 学院主动释放污染源扩招觉醒者（大真相）——以备战为名，实际在喂养卵
- 死亡螺旋：觉醒者越多 → 回响与卵共振越强 → 卵加速 → 学院更激进扩招 → 循环加速

### 个人级——回响体系
- 回响：异世界碎片与人类意识共振产生的能力，形态因人而异（性格/经历/渴望决定）
- 使用时短暂"听到/看到"异世界片段；过度使用加深连接，可能被侵蚀人格
- 不设等级阶梯，以"能做什么"和"代价"体现强弱
- 代价天花板：过度使用者可能成为异世界重新入侵的通道
