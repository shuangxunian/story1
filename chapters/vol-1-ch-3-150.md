# 第三章：回声（扩展版·150节）

## 总述
林时开始接触"对立面"——负向觉醒者不全是失控的怪物。学院不是唯一的答案，敌人不全是坏人。
核心：认知颠覆 → 地下组织接触 → 医院线 → 陆沉危机 → 陈渡真相 → 苏晚 → 顾铮松动 → 贺鸣崩溃 → 首次背叛。

场景比例：社会外勤45% / 学院内30% / 地下组织25%

衔接点（第二章065节状态）：
- 卵脉搏4.6秒，持续减速中
- 林时双通道12秒/-4%，被动场可辨4米
- 陆沉共振阶段，声音外移至西南方向，强度6
- 苏晚三次接触，白发灰化5厘米，表达"加入"意愿
- A-12源头在学院内部已确认，顾铮揭示卵是信标
- 台阶六人组形成（林时/贺鸣/陆沉/陈磊/赵晴/宋清）

- 015 [新] → 出发当天清晨的紧张感。公交上林时观察城市——普通人的早高峰与自己即将做的事之间的割裂感

- 036 [现023] → 许岚布置任务——市第三人民医院住院部出现持续性情绪异常，A-2负责实地调查，林时和沈默以病人家属身份潜入
- 037 [现024] → 林时和沈默带着藏有扫描仪的水果篮坐公交前往医院，途中讨论"静"对医疗仪器的潜在影响
- 038 [新] → 到达医院。林时第一次近距离感知大量普通病人的情绪波动——焦虑、恐惧、烦躁。被动场里这些不是回响，但密度大到让人头疼

- 041 [现026] → 追踪至地下二层太平间，确认波动来自一具未被登记的觉醒者遗体，回响呈机械式规律释放

- 045 [新] → 回总部的路上经过一片居民区。林时想：医院里那些情绪恶化的病人，出院回家后呢？有没有人在追踪后续
- 046 [现030] → 向许岚汇报。许岚揭示后勤处理组的历史和余烬案例的积压问题，要求林时把发现如实写进报告

- 050 [新] → C段收束。训练日常+备忘录反思："余烬"和"盲区"——活着没被找到，死了还在伤人。学院的网到底漏了多少

## F. 苏晚的影子（第91-105节）

- 091 [现057] → 林时第三次去城西找陈渡。陈渡提到有一个"很特别的"负向觉醒者来过——黑色光刃，来了又走了
- 092 [现058] → 陈渡讲苏晚来过两次。第一次待了两天，第二次只待了几小时。"她还没准备好留下来——怕控制不住伤到我们"
